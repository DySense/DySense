class SensorController

Public Methods

    add_sensor(SensorClient new_sensor) - validates that sensor name is unique and adds to list of sensors.
    
    remove_sensor(int sensor_id) - search through sensor_clients and remove one that has matching id. 
    
    handle_command(str command, int sensor_id) - finds sensor client tells it to send command.

    SensorClient gps_source_client (this is a normal sensor which inherits from base sensor)
    
    SensorClient gps_broadcast_client (this gets time/position/info and broadcast out on the presibed port)
